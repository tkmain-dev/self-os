## 主要機能

1. **デイリーページ** (`/daily`) — スケジュールタイムライン、習慣トラッカー、日記
2. **カレンダー** (`/calendar`) — 月・週・日ビューで予定と目標を表示
3. **目標管理** (`/goals`) — WBS ガントチャート（Epic > Story > Task > Subtask）
4. **ウィッシュリスト** (`/wishlist`) — 買いたいもの / やりたいことの2タブ管理
5. **管理ページ**（歯車ボタン） — Feature Request 管理モーダル
